۶. Pydantic AI — Agents

۶.۱ analysis_agent

from pydantic import BaseModel
from pydantic_ai import Agent

class MeetingAnalysis(BaseModel):
    title: str
    summary: str
    key_points: list[str]
    decisions: list[str]
    tasks: list[MeetingTask]

analysis_agent = Agent(
    'google:gemini-2.5-flash',
    output_type=MeetingAnalysis,
    system_prompt='تحلیل transcript جلسه فارسی. فقط بر اساس متن.',
)

۶.۲ RAG — رفتار هوشمند

  • سلام / hello: بدون جستجو در transcript — پاسخ کوتاه و دعوت به پرسش درباره جلسه
  • سوال جلسه: Chroma top-4 → context داخلی برای LLM (نمایش خام transcript در UI نیست)
  • پاسخ: فارسی طبیعی با نسبت‌دادن («علی گفت…») + footnote کوتاه در صورت نیاز
# تشخیص small talk
is_small_talk("hello") → chitchat_agent, sources=[]

# سوال واقعی
chunks = chroma.query(meeting_id, question)
rag_agent.run(context + question) → RagAnswer
Jira create از API route مستقیم است. Issueها به انگلیسی (title_en, context_en) برای خوانایی LTR در Jira — UI همچنان فارسی.

قواعد RTL/LTR:

  • html lang="fa" dir="rtl" — متن فارسی راست‌چین
  • کد، env، URL، نام API در dir="ltr"
  • نمودارهای Mermaid در wrapper با direction: ltr

صفحه /meeting/[id] داده را در سرور Astro از FastAPI می‌گیرد (نه فقط client JS) — رفع مشکل گیر کردن روی «در حال بارگذاری».
